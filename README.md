# LinZY1011 GitHub Pages 博客

这是一个直接部署到 `https://linzy1011.github.io/` 的静态个人博客站点，当前只保留：

- 首页简介
- 实验报告博文

## 目录结构

```text
linzy1011.github.io/
├─ index.html
├─ styles.css
├─ README.md
├─ .nojekyll
└─ posts/
   └─ lab2-report.html
```

## 本地预览

在仓库根目录运行：

```bash
python -m http.server 8080
```

然后访问 [http://localhost:8080/](http://localhost:8080/)。
