本地预览

最快的验证方式

在仓库根目录执行下面的命令，即可按最终部署目录结构预览页面。

python -m http.server 8080

浏览器访问 http://localhost:8080/，确认首页和文章页都能打开。

发布路径

推送到默认分支后即可作为主页仓库发布

当前仓库名为 linzy1011.github.io，因此它对应 GitHub Pages 用户主页地址。 将站点文件保存在默认分支根目录，有利于降低部署复杂度。 

git add index.html styles.css posts/lab2-report.html README.md .nojekyll
git commit -m "feat: launch personal blog on GitHub Pages"
git push origin main
